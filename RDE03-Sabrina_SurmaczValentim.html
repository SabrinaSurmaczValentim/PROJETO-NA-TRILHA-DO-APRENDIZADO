TECNOLOGIAS PARA DESENVOLVER UM SITE OU UMA PÁGINA - RESUMO

HTML 5
Linguagem de Marcação de Hipertexto, responsável por estruturar o conteúdo de um site, como texto, tabela, imagem e vídeo.
CSS 3
Responsável pela estilização de conteúdo como, cores, fontes, tamanho, sombras e posicionamento.
JS (JAVA SCRIPT)
Responsável pela interatividade, parte lógica, como, menus interativos, validação, popups e animação.

PÁGINA WEB E WEBSITE

  • Página web ou Webpage é um arquivo onde você pode inserir textos, imagens, tabelas e incorporar planilhas, apresentações, vídeos com ligações de hipertexto que permitem a navegação de uma página, ou seção, para outra.
  • Websites ou Sites são uma coleção de páginas web armazenadas em uma pasta ou em subpastas relacionadas de um servidor web.

SERVER-SIDE vs CLIENT-SIDE

A internet funciona em uma arquitetura Cliente-Servidor. O cliente faz uma requisição ao servidor e esse retorna para o cliente.

  • Server-Side: precisa ter tecnologias como, linguagem de programação, banco de dados, para atender o cliente.
  • Client-Side: possui tecnologias como, HTML, CSS e JS.

PROFISSIONAIS

  • Front-End: trabalha com HTML, CSS e Java Script, na parte do Client-Side.
  • Back-End: trabalha na parte do Server-Side
  • FullStack: trabalha como Front-End e Back-End.

DEFINIÇÃO DE HTML E TAGS

HTML - HIPERTEXT MARKUP LANGUAGE

É uma Linguagem de Marcação de Hipertexto baseada em tags, criada na década de 1990 por Tim Berners-Lee para publicação de conteúdo em hipertexto, tais como, texto, imagem, áudio e vídeo, e que podem ser vinculados através de hiperlinks.

TAGS

As tags definem ao navegador a marcação de conteúdo. As tags tem funcionalidades específicas.Sem o uso de tags o navegador exibe um texto plano.

Sintaxe:

As tags podem possuir atributos, ou seja, são parâmetros que definem algum tipo de informação extra ao navegador.

MARCAÇÃO DE QUEBRA DE LINHAS

As tags de quebra de linha produzem uma quebra de linha em um texto. É útil para escrever poemas ou um endereço, onde a divisão de linha é significante.

Observação:

  • A quebra do parágrafo é maior que a quebra de linha.
